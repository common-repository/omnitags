== Description ==

Omnitags simplifies the insertion of scripts and meta data in the source code, such as SEO meta data, analytics scripts, comments and live chat services, CRM tracking, marketing automation, widgets, ...

== Installation ==

Upload the Omnitags plugin to your Wordpress website, activate it, open the Omnitags admin menu, then enter values for the services your need to include in your website.
